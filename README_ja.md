YouTubeの動画に「興味なし」ボタンを設置する Tampermonkey スクリプトです。  
トップページと視聴ページの両方で動作します。

ボタンは2つあり、
- 左側のボタンは「興味なし」を押すだけ
- 右側のボタンは「興味なし」→「理由を教えて下さい」→「この動画を見たことがある」まで一気に押す
  
という違いがあります。YouTubeのおすすめを調整するためにお使いください。

## 視聴履歴削除ボタン
興味なしボタンに加えて、再生履歴のページで各動画に削除ボタンが付きます。
