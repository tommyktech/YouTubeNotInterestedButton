YouTubeの動画に「興味なし」ボタンなどを設置する Tampermonkey スクリプトです。  
トップページと視聴ページの両方で動作します。

# 設定画面

<img width="577" height="383" alt="image" src="https://github.com/user-attachments/assets/e8f4a2a7-6d88-4b56-bd63-c47c9f7ace90" />

ボタンは5つあり、
- 興味なし
- 興味なし → 理由を教えて下さい →　この動画を見たことがある
- 興味なし → 理由を教えて下さい →　この動画は好みではない
- チャンネルをおすすめに表示しない
- 視聴履歴を削除 (視聴履歴ページ)
  
のボタンのON/OFFをそれぞれ設定できます。


## 視聴履歴削除ボタン
興味なしボタンに加えて、再生履歴のページで各動画に削除ボタンが付きます。
